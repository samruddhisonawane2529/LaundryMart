# Laundry Mart

A simple static HTML page for Laundry Mart (Laundry Wallah) that showcases laundry services, pricing, and a basic booking form.

## Files

- `Index.html` — main website page
- `washing_machine.png` — image shown on the page (if present)

## How to open

1. Open `Index.html` in your browser.
2. Or right-click it in VS Code and choose "Open with Live Server" if you have the Live Server extension.

## What it includes

- A welcome heading and description
- A services list
- A price list table
- A booking form with name and email fields

## Notes

- This is a plain HTML page with no backend processing.
- The submit button on the form will refresh or submit the page unless connected to a server.
